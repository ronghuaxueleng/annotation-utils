# annotation-utils
#### 介绍

工具功能包括

1、提取java类和方法以及参数的注解

2、给java类和方法添加注解

3、给字段添加注解

4、提取java类中的字段

5、根据给定的对象给类、方法和字段添加属性

#### 使用说明

maven引入：

```xml
<dependency>
  <groupId>io.github.ronghuaxueleng</groupId>
  <artifactId>annotation-utils</artifactId>
  <version>${lastVersion}</version>
</dependency>
```
